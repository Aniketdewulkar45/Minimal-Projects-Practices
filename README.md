# Minimal-Projects-Practices

A collection of **quick and simple projects** created in under an hour each, designed for **hands-on learning and practice**. This includes **two HTML projects** and **one Java project**.

---

## Projects Included

1. **HTML Project 1** – Simple webpage with basic structure and navigation.  
   **Files:** `about.html`, `contact.html`, `home.html`, `login.html`, `register.html`

2. **HTML Project 2** – Another webpage with similar structure for practice.  
   **Files:** `about.html`, `contact.html`, `home.html`, `login.html`, `register.html`

3. **Java Project** – Simple Java program demonstrating core concepts.  
   **File:** `xo.java`
